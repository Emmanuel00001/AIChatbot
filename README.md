# AIChatbot
🌐 **Live Demo:** https://aichatbot-cwhu.onrender.com
A web-based AI chatbot built with Python and Flask featuring a modern chat interface.
## 🔍 What it does
- Accepts user messages through a clean web interface
- Processes messages and returns intelligent responses
- Runs entirely in the browser — no app installation needed
## 🛠️ Built With
- Python
- Flask
- HTML, CSS, JavaScript
## ⚙️ How to Run
1. Clone the repository
git clone https://github.com/Emmanuel00001/AIChatbot.git
2. Install dependencies
pip install flask
3. Run the app
python app.py
4. Open your browser and go to
http://127.0.0.1:5000
